## 一、简介

1. `BoxCore-LNMP` 是一款自动安装`LNMP`服务环境的一键部署工具，包含`Nginx`、`MySQL`和`PHP`
2. 目前仅在CentOS 6.5 minimal 安装（32/64位均可）测试过
3. php的依赖包均通过yum进行安装

## 二、安装和使用

1、安装方法

通过Git下载（推荐）

	yum -y install git
	git clone https://github.com/boxcore/lnmp.git
	cd lnmp && chmod 755 *.sh && dos2unix lnmp.sh && ./lnmp.sh
